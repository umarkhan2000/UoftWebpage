Software Documentation

Logging into Mathlab:

Mac OS comes with the application "terminal". Simple click on it to run. Login to mathlab by typing:

ssh utorid@mathlab.utsc.utoronto.ca. Enter your password.

Windows - you will need a SSH client. PuTTY is a good free one. You can download and install here: Putty . Use the same command as above to login to mathlab:

ssh utorid@mathlab.utsc.utoronto.ca.
